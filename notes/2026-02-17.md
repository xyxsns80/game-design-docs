### 主题1：借鉴《纸上谈亲》的设计理念

#### 讨论内容

1. **趣味文本的重要性**
   - 暴击/击杀/胜利文本应该随机化
   - 幽默感能让重复游玩不枯燥
   - 已实现于《肉鸽征途》

2. **信息揭示机制**
   - 敌人信息不应该一开始全部显示
   - 分阶段揭示：数量 → 属性 → 详情
   - 已实现于《肉鸽征途》

3. **继承系统（待讨论）**
   - 章节间保留部分进度
   - 让玩家感觉到成长积累

### 主题2：可持续盈利游戏的核心特征

#### 问题
什么样的游戏是一款可以持续盈利的好游戏？他们应该具有什么特点？

#### 结论

**核心公式：**
```
可持续盈利 = 高复玩性 × 多层变现 × 碎片化友好 × 自传播能力
```

**关键特征：**
1. **高复玩性** - 肉鸽、随机、收集、排行榜
2. **多层变现** - 广告+小额+中额+大额付费
3. **碎片化友好** - 单局3-5分钟，随时暂停
4. **自传播能力** - 截图有趣、话题性、挑战性

**适合小团队的方向：**
- 文字模拟（成本低）
- 肉鸽卡牌（复玩性高）
- 放置挂机（变现能力强）
- 超休闲游戏（传播快）

**最佳公式：**
```
低成本高收益 = 文字/简单美术 + 肉鸽机制 + 广告+内购 + 社交传播
```

详细分析见：[docs/mechanics/可持续盈利游戏特征.md](../docs/mechanics/可持续盈利游戏特征.md)

### 主题3：文字游戏与其他游戏的设计差异

#### 问题
文字游戏和其他游戏的从设计而非表现层面，主要区别是什么？

#### 结论

**五大核心差异：**

| 维度 | 文字游戏 | 其他游戏 |
|------|----------|----------|
| 信息载体 | 文字（抽象符号） | 图像/声音（具象感知） |
| 想象空间 | 玩家自行脑补 | 开发者定义 |
| 核心交互 | 选择驱动 | 操作驱动 |
| 内容核心 | 叙事+数值 | 沉浸+体验 |
| 迭代速度 | 极快（改文字） | 慢（改资产） |

**文字游戏的设计要点：**
1. 文案即核心（每句话都要有趣）
2. 选择即玩法（每个选择要有意义）
3. 数值即深度（属性、装备、概率）
4. 分支即内容（多结局、隐藏内容）
5. 留白即想象（不要写太满）

**文字游戏公式：**
```
好玩的文字游戏 = 有趣文案 + 有意义选择 + 深度数值 + 想象空间
```

详细分析见：[docs/mechanics/文字游戏设计差异.md](../docs/mechanics/文字游戏设计差异.md)
