# 现代日常化方案

> 目标：找出每周甚至每天都能感受到的现代生活内容

---

## 一、什么是"现代日常"？

### 标准

| 标准 | 说明 |
|------|------|
| **高频** | 每周甚至每天发生 |
| **现代** | 互联网时代特有 |
| **大众** | 几乎所有人都在经历 |
| **有共鸣** | 有吐槽点/话题点 |

---

## 二、现代日常方案

### 🏆 方案1：《手机的一天》

**高频程度：⭐⭐⭐⭐⭐（每时每刻）**

| 维度 | 内容 |
|------|------|
| **行为** | 玩手机、刷APP |
| **背景** | 2020年代的手机生活 |
| **为什么大众** | 现代人离不开手机 |

**核心玩法：**
```
模拟一天的手机使用体验

1. 早高峰
   - 闹钟响了，刷5分钟再起
   - 通勤路上刷抖音/微博
   - 微信群消息99+

2. 工作中
   - 偷偷摸鱼刷手机
   - 被领导发现
   - 微信工作群@你
   - 朋友圈屏蔽同事

3. 晚间
   - 躺床上刷手机
   - 刷到停不下来
   - 本来只想看5分钟，结果2小时过去了
   - 睡前最后一件事是看手机

4. 特色事件
   - 收到垃圾短信
   - 外卖小哥打电话
   - 朋友圈看到前任动态
   - 微信群吵架
   - APP弹广告
   - 手机没电了
   - 网络卡了
```

**为什么好：**
- ✅ 每个人每天都在做
- ✅ 有强烈的"被监控感"（手机比你还懂你）
- ✅ 有吐槽点（APP广告、微信群、算法推荐）

### 🏆 方案2：《点外卖》

**高频程度：⭐⭐⭐⭐⭐（每天/每周）**

| 维度 | 内容 |
|------|------|
| **行为** | 点外卖、等外卖、吃外卖 |
| **背景** | 外卖时代的日常 |
| **为什么大众** | 现代人经常点外卖 |

**核心玩法：**
```
1. 点餐阶段
   - 选择困难（吃什么？）
   - 凑满减
   - 看评价
   - 用优惠券

2. 等待阶段
   - 看骑手位置
   - 骑手绕路了
   - 骑手找不到
   - 配送超时

3. 特色事件
   - 外卖撒了
   - 送错了
   - 少了东西
   - 骑手打电话问路
   - 备注被无视
   - 汤洒了
   - 到了发现不好吃
```

**为什么好：**
- ✅ 高频（很多人每天点）
- ✅ 有强烈的共鸣（选择困难、等餐焦虑）
- ✅ 有吐槽点（骑手、商家、平台）

## 四、首选方案详细设计

### 🏆 方案A：《手机的一天》

**核心概念：** 你扮演一个现代人，体验一天的手机生活

```
开局：早上7点，闹钟响了
目标：度过一天，控制手机使用时间
挑战：各种诱惑让你忍不住看手机

时间线：
07:00 闹钟响了 → 刷5分钟？还是直接起？
08:00 通勤中 → 刷手机？还是发呆？
09:00 工作中 → 微信响了，看一眼？
12:00 午休 → 刷短视频？
18:00 下班 → 继续刷？
23:00 睡前 → 再看5分钟？
02:00 凌晨 → 怎么还在刷？！

结局类型：
- 自律达人（控制住了）
- 手机成瘾（没控制住）
- 意外情况（手机丢了/没电了）
```

---

### 🏆 方案B：《点外卖》

**核心概念：** 模拟点外卖的完整体验

```
核心循环：
饿了 → 选餐（选择困难）→ 下单 → 等待（焦虑）→ 收到 → 吃饭

每个环节都有事件：
选餐：看评价、凑满减、选择困难
等待：看骑手位置、配送超时、骑手找不到
收到：撒了、送错了、少了东西
吃饭：好吃/难吃/和图片不符

有趣元素：
- 商家名字和图片不符
- 评价有猫腻
- 骑手的故事
- 备注的互动
```

### 最终推荐

```
🏆 《手机的一天》
= 玩手机（行为）× 互联网时代（背景）× 控制不住自己（荒诞）

= 点外卖（行为）× 外卖时代（背景）× 各种意外（荒诞）
```
