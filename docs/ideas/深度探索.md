## 四、人际关系方向

### 方向1：《处理人际关系的一天》

| 维度 | 内容 |
|------|------|
| **核心** | 处理各种人际关系 |
| **对手盘** | 各种人（各有各的想法） |
| **反馈** | 关系变好/变差 |
| **高频** | 每天 |

**具体场景：**

```
1. 和领导的关系
   - 领导布置任务，你怎么回应？
   - 领导批评你，你怎么反应？
   - 领导画饼，你信不信？

2. 和同事的关系
   - 同事甩锅，你怎么办？
   - 同事八卦，你参与吗？
   - 同事求助，你帮吗？

3. 和家人的关系
   - 父母催婚，怎么回应？
   - 婆媳/岳婿关系
   - 和另一半的日常

4. 和朋友的关系
   - 朋友借钱
   - 朋友吐槽
   - 朋友背叛
```

---

## 八、深度设计

### 🏆 《职场生存日记》

```
公式 = 职场（场景）× 领导/同事（对手）× 升职/被裁（反馈）

核心循环：
1. 每天发生事件
2. 选择如何应对
3. 影响各方关系
4. 影响职业发展

对手盘：
- 领导（决定你的升迁）
- 同事（可能竞争/合作）
- 客户（可能刁难）
- 竞争对手（可能使坏）

特色事件：
- 领导画饼
- 同事甩锅
- 站队问题
- 背锅事件
- 被迫加班
- 年终评优
- 裁员风波
- 35岁危机

有趣元素：
- 不同选择导致不同结局
- 同样的选择，不同人反应不同
- 职场"潜规则"
- 意外的升职/被裁
```

---

### 🏆 《人际关系大师》

```
公式 = 社交（场景）× 各种人（对手）× 关系变化（反馈）

核心循环：
1. 遇到社交场景
2. 选择如何应对
3. 影响关系
4. 关系影响后续

对手盘：
- 家人（各有性格）
- 朋友（各有立场）
- 同事（各有利益）
- 陌生人（各有目的）

特色事件：
- 朋友借钱
- 亲戚八卦
- 同事八卦
- 朋友圈点赞
- 群聊发言
- 聚会应酬

有趣元素：
- 说错话得罪人
- 意外获得好感
- 关系的微妙变化
- 朋友圈的"暗战"
```

## 九、终极推荐

### 🏆 首选：《职场生存日记》

**为什么：**

1. **最高频** - 每个打工人每天都要面对
2. **强对手盘** - 领导/同事/客户都是"对手"
3. **明确反馈** - 升职/加薪/被裁/降薪
4. **强共鸣** - 打工人群体最大
5. **强话题** - 职场话题永远热门
6. **有策略** - 不同选择不同结局

**对标成功案例：**
- 《纸上谈亲》= 相亲 + 各种奇葩对象
- 《职场生存》= 职场 + 各种奇葩领导/同事
