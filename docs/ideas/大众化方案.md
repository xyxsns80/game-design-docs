## 二、大众化小切口方案

### 🏆 方案1：《高考模拟器》

**大众化程度：⭐⭐⭐⭐⭐**

| 维度 | 内容 |
|------|------|
| **行为** | 高考备考、考试 |
| **背景** | 高三那年、夏天、压力 |
| **为什么大众** | 几乎每个人都经历过高考，或知道高考 |
| **文化共识** | "高考改变命运"是社会共识 |

**核心玩法：**
```
1. 开局设定
   - 省份（决定难度）
   - 家庭背景（影响资源）
   - 天赋属性（智力、毅力、运气）

2. 核心循环
   - 备考（选择学习方式）
   - 模考（看成绩变化）
   - 高考（最终一搏）
   - 填志愿（选择学校）
   - 结局（录取结果）

3. 特色事件
   - 模考失利
   - 老师谈话
   - 父母施压
   - 同学竞争
   - 恋爱分心
   - 考前失眠
   - 考场意外
```

**为什么好：**
- ✅ 几乎人人都经历过
- ✅ "高考"是文化符号
- ✅ 有明确的成功/失败标准
- ✅ 容易触发回忆
- ✅ 结果有荒诞感（意外录取/落榜）

### 🥈 方案2：《面试模拟器》

**大众化程度：⭐⭐⭐⭐⭐**

| 维度 | 内容 |
|------|------|
| **行为** | 面试求职 |
| **背景** | 经济环境、行业、季节 |
| **为什么大众** | 几乎每个人都面过试 |

**核心玩法：**
```
1. 开局设定
   - 学历、专业、经验
   - 目标岗位/公司

2. 核心循环
   - 投简历
   - 收到面试邀请
   - 面试过程（问答选择）
   - 等结果
   - 入职/被拒

3. 特色事件
   - HR问"你的缺点是什么"
   - 面试官很凶/很和蔼
   - 面试迟到
   - 紧张说错话
   - 被问薪资期望
   - 遇到奇葩面试官
```

### 🥉 方案3：《买房模拟器》

**大众化程度：⭐⭐⭐⭐**

| 维度 | 内容 |
|------|------|
| **行为** | 看房、买房 |
| **背景** | 房价走势、政策变化 |
| **为什么大众** | 买房是人生大事，即使没买过也关注 |

**核心玩法：**
```
1. 开局设定
   - 城市、预算、家庭情况

2. 核心循环

   - 纠结买不买

3. 特色事件
   - 房东临时涨价
   - 刚卖掉就没房了
   - 发现房子有坑
   - 房价突然涨/跌
   - 父母出钱/不出钱
```

### 方案4：《过年回家》

**大众化程度：⭐⭐⭐⭐⭐**

| 维度 | 内容 |
|------|------|
| **行为** | 过年回家、应对亲戚 |
| **背景** | 春节、冬天 |
| **为什么大众** | 中国人都要过年 |

**核心玩法：**
```
1. 开局设定
   - 年龄、职业、婚恋状况

2. 核心循环
   - 回家路上
   - 应对亲戚提问
   - 年夜饭
   - 发/收红包
   - 走亲戚

3. 特色事件（七大姑八大姨经典问题）
   - "有对象了吗？"
   - "工资多少？"
   - "什么时候结婚？"
   - "买房了吗？"
   - "生孩子了吗？"
   - 攀比："你表哥都当经理了"
```

**为什么好：**
- ✅ 极度大众化
- ✅ 有强烈的文化共鸣
- ✅ 有吐槽点（亲戚提问）
- ✅ 有幽默感（可以很荒诞）

| 维度 | 内容 |

   - 忘记打卡
