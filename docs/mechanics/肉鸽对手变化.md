# 肉鸽核心问题：对手要能换

> 关键洞察：纸上谈亲的对象每次都换，所以是肉鸽。职场生存的老板/同事不会经常换，所以更像叙事游戏。

---

## 一、问题分析

### 《纸上谈亲》的肉鸽本质

```
每局相亲：
1. 随机生成新的相亲对象（新对手）
2. 新的标签组合（新属性）
3. 新的隐藏信息（新未知）
4. 新的选择和结果（新体验）

核心：对手每次都换 → 每局都是新体验
```

### 《职场生存》的问题

```
同一个公司：
1. 老板是同一个
2. 同事是同一批
3. 每天的事件类似
4. 变化不够大

问题：对手不换 → 像叙事游戏，不像肉鸽
```

---

## 二、肉鸽的核心要素

### 什么是肉鸽？

```
肉鸽 = 每局都是新体验

关键要素：
1. 随机生成内容（对手/关卡/事件）
2. 每局体验不同
3. 有积累/成长（跨局）
4. 有失败重开机制
```

### 对手变化的类型

| 类型 | 说明 | 示例 |
|------|------|------|
| **完全随机** | 每局全新对手 | 《纸上谈亲》 |
| **池子随机** | 从池子中随机抽 | 《杀戮尖塔》怪物池 |
| **条件触发** | 满足条件出现新对手 | 游戏后期新敌人 |
| **组合随机** | 同样对手+不同属性 | 同类怪不同数值 |

## 三、如何让对手能换？

### 方案1：场景切换

**《面试模拟器》**

```
每局是不同的面试：
- 公司A：互联网大厂，面试官A1
- 公司B：创业公司，面试官B1
- 公司C：外企，面试官C1

对手每次都换：不同的公司、不同的面试官
```

### 方案2：时间线推进

**《相亲之旅》**（扩展版纸上谈亲）

```
不是一次相亲，而是相亲之旅：
- 相亲1 → 相亲2 → 相亲3 → ... → 结局

每局可以相亲多次，每次遇到不同的人
```

### 方案3：场景池

**《社交场合模拟器》**

```
从场景池中随机抽取：
- 今天是公司聚餐
- 今天是朋友聚会
- 今天是相亲
- 今天是家庭聚餐

每个场景有不同的NPC
```

### 方案4：事件池

**《打工人的一天》**

```
每天随机事件：
- 今天遇到奇葩客户
- 今天领导发飙
- 今天同事请假
- 今天系统崩溃

事件池随机抽取，每次不同
```

```
肉鸽游戏的核心：
1. 对手/事件可以随机生成
2. 每局体验不同
3. 可以快速重开
```

## 五、真正的肉鸽方案

### 🏆 方案1：《面试大作战》

```
核心：去不同公司面试

肉鸽要素：
- 每局是不同的公司（随机生成）
- 每个公司有不同的面试官（随机生成）
- 每个面试官有不同的风格/问题（随机生成）

对手变化：
- 公司池：大厂/创业/外企/国企/奇葩公司
- 面试官池：HR/技术/老板/奇葩面试官
- 问题池：常规/刁钻/奇葩

循环：
面试A公司 → 结果 → 面试B公司 → 结果 → ... → 最终结局（拿到offer/没拿到）

每局可以面试多个公司，每个公司都是新对手
```

### 🏆 方案2：《谈判专家》

```
核心：接不同的谈判任务

肉鸽要素：
- 每局是不同的谈判任务（随机生成）
- 每个任务有不同的对手（随机生成）
- 每个对手有不同的性格/底线（随机生成）

对手变化：
- 谈判场景：薪资/合同/纠纷/交易
- 谈判对手：老板/客户/房东/前任

循环：
接任务A → 谈判 → 结果 → 接任务B → ...
```

### 🏆 方案3：《客服大战》

```
核心：应对不同的客户

肉鸽要素：
- 每局是不同的客户（随机生成）
- 每个客户有不同的问题/性格（随机生成）
- 每个客户有不同的满意度标准（随机生成）

对手变化：
- 客户池：正常客户/奇葩客户/暴躁客户/骗子
- 问题池：退货/投诉/咨询/纠纷

循环：
接单A → 处理 → 评价 → 接单B → ...
```

### 🥈 方案4：《社交场》

```
核心：不同的社交场合

肉鸽要素：
- 每局是不同的社交场合（随机抽取）
- 每个场合有不同的NPC（随机生成）
- 每个NPC有不同的性格/关系（随机生成）

场景池：
- 公司聚餐
- 朋友聚会
- 相亲
- 家庭聚餐
- 同学聚会
- 行业聚会

循环：
场景A → 社交 → 结果 → 场景B → ...
```

## 七、首选方案详细设计

### 🏆 《面试大作战》

**核心概念：** 去不同公司面试，每次都是新的对手

```
公式 = 面试（行为）× 随机公司/面试官（可换对手）× offer结果（反馈）

开局设定：
- 你的标签：学历、专业、经验、期望薪资
- 目标：拿到几个offer/拿到什么级别的offer

核心循环：
1. 浏览职位（随机生成）
2. 投简历
3. 收到面试邀请
4. 面试过程（与面试官博弈）
5. 等结果
6. 重复或结束

对手变化：
- 公司类型：大厂/创业/外企/国企/奇葩公司
- 面试官：HR/技术/老板/奇葩面试官
- 面试官性格：和蔼/严厉/奇葩/阴阳怪气
- 问题类型：常规/刁钻/奇葩/压力面

特色事件：
- 面试官迟到
- 问奇葩问题
- PUA式面试
- 当场给offer
- 面试通过但薪资谈崩
- 发现面试官是前任

结局：
- 拿到多个offer（成功）
- 拿到一个offer（普通）
- 没拿到offer（失败）
- 拿到奇葩offer（荒诞）
```

## 八、总结

### 核心原则

```
肉鸽游戏 = 对手必须能换 + 每局体验不同
```

### 最终推荐

```
🏆 《面试大作战》

为什么：
1. 对手每次都换（不同公司/面试官）
2. 高频（很多人经常面试/跳槽）
3. 有对手盘（面试官决定录用与否）
4. 明确反馈（拿到offer/被拒绝）
5. 有策略深度（不同面试官不同应对）
6. 有荒诞感（奇葩公司/奇葩问题）
```
