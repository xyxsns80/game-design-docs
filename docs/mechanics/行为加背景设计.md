# 行为 + 背景：多维代入感设计

> 讨论：除了行为本身，社会背景、时代背景、天气等环境因素也是共同体验，能否结合？

---

## 一、核心洞察

### 用户的发现

**行为**是体验的一部分，但**背景**同样是共同体验：

| 背景 | 说明 | 是否共同体验 |
|------|------|--------------|
| 社会背景 | 经济形势、就业环境、社会压力 | ✅ 大家都身处其中 |
| 时代背景 | 80年代、90年代、00年代 | ✅ 每代人都有共同记忆 |
| 天气/季节 | 雨天、夏天、冬天 | ✅ 每个人都经历过 |
| 大事件 | 疫情、奥运会、房价暴涨 | ✅ 大家都经历过 |

**关键洞察**：
> 背景不是行为，但背景是**共同体验**，同样可以产生代入感！

---

## 二、行为 vs 背景

### 定义

| 概念 | 定义 | 示例 |
|------|------|------|
| **行为** | 玩家在游戏中做的事 | 相亲、面试、考试 |
| **背景** | 行为发生的环境和时代 | 经济下行、疫情期间、夏天 |

### 对代入感的贡献

| 维度 | 行为 | 背景 |
|------|------|------|
| **代入方式** | "我做过这个！" | "我经历过这个时代！" |
| **共鸣来源** | 个人经历 | 集体记忆 |
| **情感类型** | 具体的 | 氛围的 |
| **记忆触发** | "对对对，我也有过" | "那时候真的就是这样" |

### 示例对比

**纯行为设计：**
```
你在咖啡厅相亲，对方迟到了10分钟...
→ 代入感：我也相过亲，我也等过人
```

**行为+背景设计：**
```
2023年的冬天，经济下行，你在咖啡厅相亲
对方迟到了10分钟，说"不好意思，刚被裁员，处理了点事情"
→ 代入感：我也相过亲 + 那时候真的很难 + 冬天的氛围
```

## 三、背景的类型

### 1. 社会背景

| 背景 | 共鸣人群 | 示例事件 |
|------|----------|----------|
| 经济下行 | 所有人 | 裁员、降薪、消费降级 |
| 就业难 | 年轻人 | 卷学历、海投简历 |
| 房价高 | 打工人 | 买房焦虑、租房坑 |
| 内卷 | 所有人 | 996、KPI、35岁危机 |
| 躺平 | 年轻人 | 不想努力了 |

### 2. 时代背景

| 时代 | 共鸣人群 | 共同记忆 |
|------|----------|----------|
| 80年代 | 60-70后 | 改革开放、下海、分房 |
| 90年代 | 70-80后 | 下岗潮、港台流行、BP机 |
| 00年代 | 80-90后 | 加入WTO、非典、房价起飞 |
| 10年代 | 90-00后 | 移动互联网、共享经济 |
| 20年代 | 所有人 | 疫情、远程办公、AI |

### 3. 天气/季节

| 背景 | 情感联想 | 示例 |
|------|----------|------|
| 夏天 | 燥热、躁动、青春 | 蝉鸣、高考、毕业 |
| 冬天 | 冷清、温暖、年末 | 寒风、过年、年终奖 |
| 雨天 | 忧郁、浪漫、不便 | 被淋湿、约会取消 |
| 春天 | 希望、新的开始 | 求职季、新学期 |

### 4. 大事件

| 事件 | 影响范围 | 共同记忆 |
|------|----------|----------|
| 疫情 | 全球 | 口罩、隔离、远程办公 |
| 奥运会 | 全国 | 看比赛、金牌、自豪 |
| 房价暴涨 | 全国 | 买不起、焦虑、观望 |
| 股市波动 | 投资者 | 赚钱、亏钱、躺平 |

## 四、行为+背景的组合设计

### 组合公式

```
代入感 = 行为共鸣 × 背景共鸣
```

### 组合示例

| 行为 | 背景 | 组合效果 |
|------|------|----------|
| 相亲 | 经济下行 | "没钱还相什么亲啊" |
| 面试 | 疫情期间 | "视频面试，网还卡了" |
| 加班 | 夏天 | "空调坏了还要加班" |
| 买房 | 房价暴涨 | "看着房价涨，买不买" |
| 毕业 | 就业难 | "毕业即失业" |

### 设计维度

**单一维度（行为）：**
```
你今天去相亲...
```

**双维度（行为+时代）：**
```
1998年，你今天去相亲...
→ 触发：下岗潮背景
```

**三维度（行为+时代+天气）：**
```
1998年的冬天，工厂倒闭了
你在寒风中，骑着自行车去相亲...
→ 触发：时代记忆 + 天气氛围
```

**四维度（行为+时代+天气+社会）：**
```
1998年的冬天，下岗潮席卷全国
你在寒风中骑车去相亲
对方问你：你现在在哪上班？
你说：我...刚"毕业"...
→ 触发：时代记忆 + 天气氛围 + 社会压力 + 尴尬情境
```

## 五、背景设计的价值

### 1. 增强代入感

```
纯行为："我相过亲"
行为+背景："我相过亲，而且那时候..."
```

### 2. 增加话题性

```
纯行为："这游戏讲相亲的"
行为+背景："这游戏讲疫情时候相亲的，那时候真的..."
```

### 3. 丰富内容

```
同一个行为 + 不同背景 = 不同体验

相亲 + 90年代 = 保守、父母介绍、工厂
相亲 + 10年代 = 互联网、网恋、白领
相亲 + 疫情期间 = 视频、口罩、距离
```

### 4. 降低内容创作难度

```
不需要设计新行为，换背景就是新内容：

相亲（夏天）→ 相亲（冬天）→ 相亲（疫情）
面试（经济好）→ 面试（经济差）→ 面试（AI时代）
```

## 六、设计建议

### 1. 背景作为"调味料"

```
主要：行为（相亲）
辅助：背景（经济下行、冬天、疫情期间）

背景不是主角，但让体验更丰富
```

### 2. 选择有共鸣的背景

```
好的背景：
✅ 经济下行（当下痛点）
✅ 疫情（共同记忆）
✅ 90年代（怀旧）
✅ 夏天（季节共鸣）

不好的背景：
❌ 2045年（没共鸣）
❌ 架空世界（没代入感）
```

### 3. 背景要和行为呼应

```
好的组合：
✅ 面试 + 就业难
✅ 买房 + 房价涨
✅ 相亲 + 经济差

不呼应的组合：
⚠️ 相亲 + 外星人入侵（太跳戏）
```

### 4. 背景可以触发特殊事件

```
普通相亲：
- 对方迟到
- 聊天尴尬

疫情期间相亲：
- 视频相亲，网卡了
- 不能见面，只能网恋
- 见面了，都戴着口罩
- 约会只能去人少的地方
```

## 七、具体案例

### 案例1：《疫情相亲记》

```
行为：相亲
背景：2020年疫情期间

特色事件：
- 第一次见面是视频
- 约会只能去公园
- 都戴着口罩，看不清脸
- 聊天的主题是"你们那边封了吗"
- 想牵手但要考虑距离
```

### 案例2：《98年相亲》

```
行为：相亲
背景：1998年下岗潮

特色事件：
- 父母介绍的
- 问的第一句是"你在哪个厂"
- 骑自行车去
- 没有手机，留BP机号
- "稳定"是最重要的
```

### 案例3：《内卷时代加班》

```
行为：加班
背景：2023年内卷时代

特色事件：
- 晚上10点还在公司
- 群里领导还在发消息
- 看到同事还在不敢走
- 点外卖继续干活
- 周末也要"自愿"加班
```

## 八、总结

### 核心观点

```
行为是"你在做什么"
背景是"你在什么环境下做"
两者结合 = 立体的代入感
```

### 设计公式

```
好的文字游戏 = 有共鸣的行为 × 有共鸣的背景
```

### 实践建议

1. 先确定核心行为（小切口）
2. 选择能增强代入感的背景
3. 背景触发特殊事件
4. 背景和行为要呼应
