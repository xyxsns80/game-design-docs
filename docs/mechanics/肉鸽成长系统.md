# 肉鸽成长系统设计

> 核心问题：局外成长是什么？成长后能挑战更强的对手吗？

---

## 一、肉鸽的成长类型

### 类型1：局内成长（每局内）

```
例子：《杀戮尖塔》
- 每局获得新卡牌
- 卡牌组合变强
- 但下一局清零
```

### 类型2：局外成长（跨局）

```
例子：《吸血鬼幸存者》
- 解锁新角色
- 永久升级属性
- 下一局更强

例子：《纸上谈亲》
- 后代继承标签
- 下一局开局更好
```

### 类型3：技能成长（玩家自己）

```
例子：各种游戏
- 玩家更懂策略
- 知道怎么选更好
- 但没有数值成长
```

---

## 二、《纸上谈亲》的成长系统

```
局内：
- 每次相亲可能获得更好的标签
- 结婚后后代继承标签

局外（跨局）：
- 后代继承 = 下一局开局更强
- 标签池可能扩大（解锁新标签？）

成长体现：
- 上一局获得"名校"标签
- 下一局开局就有"名校"
- 能匹配到更好的对象
```

## 三、各方向成长分析

### 面试

**局内成长：**
```
问题：面试完了就结束了，成长在哪？

可能的成长：
- 面试技巧提升？（但这是一次性的）
- 简历变好？（需要时间）

很难设计局内成长...
```

**局外成长：**
```
可能的设计：
- 解锁新技能（面试技巧）
- 解锁新标签（证书、项目经验）
- 下一局简历更好

问题：面试本身不是"可以重复提升"的事
```

**结论：成长设计困难 ⚠️**

---

### 谈判

**局内成长：**
```
问题：每次谈判是独立事件

可能的成长：
- 谈判技巧？（但谈判完就结束了）
```

**局外成长：**
```
可能的设计：
- 解锁谈判技能
- 声望提升
- 能接更难的谈判

问题：和面试一样，是独立事件
```

**结论：成长设计困难 ⚠️**

---

### 相亲（纸上谈亲模式）

**局内成长：**
```
✅ 有！
- 结婚后后代继承标签
- 每次相亲都可能获得更好的标签组合
```

**局外成长：**
```
✅ 有！
- 后代继承 = 下一局更强
- 标签池可能扩大
```

**结论：成长设计清晰 ✅**

## 四、什么题材适合成长系统？

### 成长的本质

```
成长 = 累积 + 变强 + 挑战更强

累积：获得某种永久的东西
变强：下一局/下一次更强
挑战更强：能应对更难的对手
```

### 适合成长的特征

| 特征 | 说明 |
|------|------|
| **可重复** | 同样的事可以做很多次 |
| **可累积** | 每次都能获得点什么 |
| **有等级** | 对手/任务有难度等级 |
| **可继承** | 成果可以带到下一局 |

## 六、思考：如何让面试/谈判有成长？

### 方案A：《面试成长记》

```
改成：不是一次次面试，而是一个"职业成长"故事

设定：
- 你是一个职场人
- 目标：从实习生 → 打工人 → 经理 → 总监 → CEO

每局：
- 一个职位阶段
- 经历多次面试/任务
- 获得成长
- 挑战下一阶段

成长：
- 技能点（沟通/专业/领导力）
- 简历标签（项目/证书）
- 人脉（NPC关系）

对手变化：
- 实习：HR面试
- 正式：技术面试
- 经理：老板面试
- 总监：董事会面试

继承：
- 上一局的技能点保留部分
- 下一局开局更强
```

**问题：变成了叙事游戏？**

---

### 方案B：《谈判王国》

```
改成：你是一个谈判专家，接各种任务

设定：
- 你是一个谈判顾问
- 目标：成为顶级谈判专家

每局：
- 接一个谈判任务
- 完成获得声望/技能
- 接更难的任务

成长：
- 谈判技能（威慑/说服/妥协）
- 声望等级
- 解锁新任务

对手变化：
- Lv1：菜市场大妈
- Lv2：二手车商
- Lv3：房东
- Lv4：大老板
- Lv5：跨国谈判

继承：
- 技能点保留
- 声望保留
```

## 八、新思路：成长型题材

### 🏆 方向1：《相亲进化论》（类似纸上谈亲）

```
已有成功模式，不需要重新设计

成长：
- 标签继承
- 后代更强
- 匹配更好的对象
```

### 🏆 方向2：《修仙之路》

```
为什么适合：
- 对手可换：不同境界遇到不同敌人
- 有成长：境界、功法、法宝
- 成长有用：高境界打低境界碾压

成长系统：
- 境界（练气→筑基→金丹→元婴...）
- 功法（学新功法）
- 法宝（获得法宝）
- 灵根（天赋）

每局：
- 从某个境界开始
- 修炼、战斗、渡劫
- 可能在某次失败
- 但获得的东西继承到下一局
```

### 🏆 方向3：《谈判专家》（改造后）

```
改造思路：加入成长系统

成长系统：
- 技能树（威慑/说服/妥协/心理战）
- 声望等级（接更难的任务）
- 人脉（解锁特殊任务）

每局：
- 接一个谈判任务
- 使用技能
- 获得经验/声望
- 下一局解锁新技能/任务

对手变化：
- 按难度分级
- 高难度需要高技能
```

## 九、最终推荐

### 🏆 《相亲进化论》

```
理由：
1. 对手可换 ✅（每次不同对象）
2. 成长系统 ✅（后代继承标签）
3. 成长有用 ✅（更好的标签匹配更好的对象）
4. 已验证成功 ✅（纸上谈亲）

成长设计：
局内：结婚→后代继承标签
局外：下一局开局更好

公式：
相亲（行为）× 随机对象（可换对手）× 后代继承（成长）
```

### 🥈 《谈判专家》（改造后）

```
理由：
1. 对手可换 ✅（每次不同任务/对手）
2. 成长系统 ✅（技能树/声望）
3. 成长有用 ✅（高技能接高难度）

成长设计：
局内：完成任务获得经验
局外：技能升级、解锁新任务
```

## 十、总结

### 肉鸽三要素

```
1. 对手可换 → 每局体验不同
2. 有成长系统 → 有追求
3. 成长有用 → 能挑战更强的对手
```

```

- 对手每次换 ✅
```
