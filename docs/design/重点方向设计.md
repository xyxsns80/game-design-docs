# 重点方向：《松人紧人》+《数字游民》详细设计

> 根据用户反馈，聚焦这两个方向

---

## 一、《松人紧人》详细设计

### 核心概念

```
你是一个打工人，每天都在"松"（躺平）和"紧"（内卷）之间摇摆

核心机制：
- 状态条：松 ←────────→ 紧
- 太松 → 被裁
- 太紧 → 身心崩溃
- 平衡 → 勉强活着
```

### 玩家怎么提升？

#### 局内成长

| 成长 | 说明 |
|------|------|
| **技能熟练度** | 工作技能提升（摸鱼时也能偷偷学） |
| **人脉关系** | 认识同事、领导、客户 |
| **存款** | 累积的工资 |
| **身心健康** | 压力管理 |
| **领导印象** | 影响升职/被裁 |

#### 局外成长（跨局继承）

| 成长 | 说明 |
|------|------|
| **技能点** | 每局获得，永久保留 |
| **解锁技能** | "摸鱼大师"、"卷王"、"社交达人"等 |
| **解锁职业** | 不同行业有不同玩法 |
| **解锁公司** | 不同公司文化不同 |

### 对手（多方）怎么变换？

#### 1. 领导（推你紧）

```
类型随机生成：
- 画饼型：总说公司要上市，让你多干
- PUA型：总说你不行，要证明自己
- 加班型：自己不走，你也不敢走
- 好领导：理解你，偶尔能松一下

每局随机：
- 不同领导有不同风格
- 领导心情会变化
- 领导可能换人
```

#### 2. 同事（在卷你）

```
类型随机生成：
- 卷王：每天加班到半夜
- 摸鱼王：很会偷懒但不出事
- 搬运工：只做分内事
- 小人：喜欢打小报告
- 朋友：能帮你

每局随机：
- 同事组合不同
- 同事会离职/入职
- 同事关系会变化
```

#### 3. 自己（想松）

```
自己的状态：
- 身体：太紧会生病
- 心理：太紧会崩溃
- 欲望：想有自己的生活
- 经济：需要钱

每局随机：
- 初始状态不同
- 遇到的个人事件不同
```

#### 4. 家人（要你稳）

```
类型随机生成：
- 父母：催婚、要钱、关心你
- 伴侣：需要陪伴、需要钱
- 孩子：需要钱、需要时间

每局随机：
- 家庭情况不同
- 家庭事件不同
```

### 危机是什么？

| 危机 | 松/紧导致 | 后果 |
|------|-----------|------|
| **被裁** | 太松 | 收入归零，游戏失败 |
| **身心崩溃** | 太紧 | 生病/抑郁，游戏失败 |
| **家庭危机** | 只顾工作 | 关系破裂 |
| **经济危机** | 收入不够 | 无法生活 |
| **职场危机** | 得罪人 | 被孤立/被裁 |

### 玩家玩的是什么？

```
核心体验：
1. 每天做选择（松还是紧）
2. 平衡多方压力（领导、同事、自己、家人）
3. 应对随机事件
4. 看能撑多久

每回合：
1. 看今天的状态和事件
2. 做选择（加班/摸鱼/学习/陪家人...）
3. 状态变化
4. 判定（是否出事）

结局：
- 被裁（太松）
- 崩溃住院（太紧）
- 裸辞（主动退出）
- 升职（运气好）
- 找到平衡（活得久）
- 意外暴富（运气）
```

### 事件池（肉鸽要素）

```
职场事件：
- 年终评优（紧的人有优势）
- 突然裁员（松的人危险）
- 领导换人（新领导新风格）
- 同事离职（工作更多）
- 项目上线（必须紧）
- 公司团建（必须参加）

个人事件：
- 身体报警（必须休息）
- 朋友借钱（经济压力）
- 家人生病（需要钱和时间）
- 恋爱/分手（情绪影响）
- 想辞职（心态变化）

随机事件：
- 疫情居家（工作方式变化）
- 经济下行（公司压力大）
- 行业风口（机会来了）
- 副业机会（额外收入）
```

## 二、《数字游民》详细设计

### 核心概念

```
你是一个自由职业者，可以自由选择在哪里工作、做什么项目

核心机制：
- 自由度高
- 不稳定性高
- 收入波动大
- 孤独感

目标：在自由和稳定之间找到平衡
```

### 玩家怎么提升？

#### 局内成长

| 成长 | 说明 |
|------|------|
| **技能等级** | 完成项目提升 |
| **客户关系** | 满意度影响回头客 |
| **口碑** | 影响接单 |
| **收入** | 累积存款 |
| **人脉** | 认识更多人 |

#### 局外成长（跨局继承）

| 成长 | 说明 |
|------|------|
| **技能点** | 永久技能提升 |
| **客户池** | 之前服务过的客户 |
| **解锁技能** | 新技能树 |
| **解锁地点** | 新的工作地点 |
| **解锁生活方式** | 不同的生活选择 |

### 对手（多方）怎么变换？

#### 1. 客户（给你工作）

```
类型随机生成：
- 挑剔型：要求很多，改来改去
- 大方型：给钱多，要求少
- 拖欠型：付款慢
- 长期型：稳定合作
- 坑人型：骗方案

每局随机：
- 客户池不同
- 客户需求变化
- 客户预算变化
```

#### 2. 竞争者（抢你生意）

```
类型：
- AI（越来越强）
- 低价竞争者（卷价格）
- 高手（质量更好）
- 团队（能接大项目）

每局随机：
- 竞争程度不同
- 新竞争者出现
```

#### 3. 市场（需求变化）

```
变化：
- 某技能突然火爆/冷却
- 价格波动
- 新需求出现
- 旧需求消失

每局随机：
- 市场趋势不同
- 机会不同
```

#### 4. 自己（内心挣扎）

```
状态：
- 孤独感（没同事）
- 焦虑感（收入不稳定）
- 自由感（想干嘛干嘛）
- 成就感（靠自己）

每局随机：
- 心理状态变化
- 个人事件
```

### 危机是什么？

| 危机 | 原因 | 后果 |
|------|------|------|
| **收入断档** | 接不到单 | 经济危机 |
| **技能过时** | 市场变化 | 没人要 |
| **身心崩溃** | 太拼 | 生病/抑郁 |
| **孤独感** | 长期独处 | 心理问题 |
| **被AI替代** | AI进化 | 失去竞争力 |

### 玩家玩的是什么？

```
核心体验：
1. 选择自由（在哪工作、做什么）
2. 管理不稳定（收入、客户）
3. 应对竞争（AI、其他人）
4. 寻找平衡（自由vs稳定）

每回合：
1. 选择工作地点（家/咖啡厅/旅途中）
2. 选择接什么项目
3. 完成工作
4. 处理生活事务
5. 结算收入和状态

结局：
- 回去上班（受不了不稳定）
- 财务自由（赚够了）
- 穷困潦倒（失败了）
- 找到平衡（稳定自由）
- 成为老板（组建团队）
```

### 事件池（肉鸽要素）

```
工作事件：
- 大项目来了（机会）
- 客户跑单（损失）
- 项目延期（要多做）
- 收到好评（口碑+1）
- 被坑了（损失）

市场事件：
- 某技能火爆（机会）
- AI能做了（威胁）
- 价格暴跌（收入降）
- 新平台出现（机会）

生活事件：
- 朋友来访（社交）
- 生病了（要花钱）
- 想家了（情绪）
- 遇到同行（人脉）
- 设备坏了（要花钱）

旅行事件（选择旅行时）：
- 遇到有趣的人
- 找到新客户
- 丢了东西
- 没网了
```

## 三、两个方向对比

| 维度 | 松人紧人 | 数字游民 |
|------|----------|----------|
| **场景** | 职场 | 自由职业 |
| **核心** | 状态平衡 | 自由vs稳定 |
| **对手** | 领导/同事/自己/家人 | 客户/竞争者/市场/AI |
| **危机** | 被裁/崩溃 | 收入断档/孤独 |
| **共鸣** | 打工人 | 自由职业者 |
| **成长** | 技能/人脉 | 技能/客户/口碑 |
| **结局** | 多种 | 多种 |

---

## 四、建议

### 优先做《松人紧人》

理由：
1. **共鸣更广** - 打工人群体更大
2. **话题更强** - 2024爆火热梗
3. **开发更简单** - 场景固定（公司）
4. **传播更容易** - 人人有共鸣

### 《数字游民》作为第二个项目

理由：
1. **受众较窄** - 自由职业者较少
2. **机制更复杂** - 地点、项目、客户
3. **可以复用** - 核心机制类似
