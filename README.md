# 游戏设计文档

记录我们对游戏设计的讨论和思考。

## 仓库结构

```
/docs
  /ideas          - 游戏创意想法
  /analysis       - 游戏分析（竞品、参考游戏）
  /mechanics      - 游戏机制设计
  /ui-ux          - 界面和交互设计
  /monetization   - 变现设计
/notes
  - 各种笔记和讨论记录
```

## 参考游戏

### 《纸上谈亲》
- **类型**: 文字模拟 + 肉鸽 + 牌组构建
- **平台**: 微信小游戏
- **亮点**:
  - 标签系统（稀有度）
  - 隐藏/揭示机制
  - 继承循环
  - 幽默文本
  - 轻量级设计
- **热度**: 上线后飙升25882%

### 《人生重开模拟器》
- **类型**: 文字模拟
- **作者**: 同《纸上谈亲》作者
- **特点**: 荒诞人生、幽默抽象

## 设计原则

1. **简单机制，深度体验**
2. **正反馈循环**
3. **幽默感**
4. **碎片化友好**
